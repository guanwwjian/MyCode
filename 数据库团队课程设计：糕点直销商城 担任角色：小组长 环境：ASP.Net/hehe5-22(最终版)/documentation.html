Modification instructions

In the .xml file that comes with the package, you can set a number of variables in the <settings /> tag. They are shown in the table below:

Variable name Variable type Value range Variable description
Automatical rotation      
autoRotate Boolean 0/1 Whether to automatically scroll through the images (only available when rotationKind is set to 1)
autoRotateSpeed Number any The time before the carousel autorotates to the next image.
Caption display      
useSubtitle Boolean 0/1 Whether to show the captiontext below the image (only available when rotationKind is set to 1)
useTooltip Boolean 0/1 Whether to show a tooltip on mouseOver
useSecondCaption Boolean 0/1 Whether to show a second line of caption.
Positioning      
spanX Number any Width of the carousel
spanY Number any Height of the carousel
centerX Number any The x-position of the center
centerY Number any The y-position of the center
Perspective      
distanceValue Number 0-1 The z-distance from the viewer. When this is 0 the front image will have a 100% size.
perspectiveRatio Number 0-1 How much perspective should be applied. 1 = none.
minimumscale Number 0-0.9 The size of the back image relative to the front image. If this is 0.5, the back image will be 50% of the front image's size.
turningspeed Number > 0 Speed of the rotation
rotationKind Boolean 0/1 The type of rotation. 0 will make the carousel rotate based on the _xmouse position. 1 will create buttons and navigation bar.
Visual effects
useFocalBlur Boolean 0/1 Whether to apply focal blur (1) or not (0)
focalBlurValue Number 0-1 Amount of focal blur
useMotionBlur Boolean 0/1 Whether to apply motion blur (1) or not (0)
motionBlurValue Number 0-1 Amount of motion blur
useFadeOnMouseOver Boolean 0/1 Whether to highlight image on mouseover
mouseOverAlphaValue Number 0-1 Alpha value of images that fade out on mouseover (see above)
useReflection Boolean 0/1 Whether to apply a reflection on images
reflectionAlphaValue Number 0-1 Alpha value of the reflection

Every image also has its own properties. They are listed in the table below:

Variable name Variable type Value range Variable description
Image      
ImageURL String any URL of the image to load
Linkage      
linkData String any Either a URL to visit when clicked, a frame number, a frame label or a movie to load.
linkType String "URL", "frameNumber", "frameLabel" or "loadMovie" A command to tell what to do with the linkData and linkTarget. See below for examples.
linkTarget String A html target or the movieclip from which linkData is defined. Telling what the linking command applies to. See below for examples.
Caption text      
captionText String any The first caption text of the image
captionText2 String any A second line of captiontext, only visible when useSecondCaption is set to 1 (see above)
Button activity      
buttonEnabledWhenInFront Boolean 0/1 Whether to enable the button when the image is in front (if rotationKind = 1). This can be used to load a .swf that you'd like to keep it's own button actions.

Linking examples

The example below would visit http://www.google.com in a new browser window.
linkData="http://www.google.com" linkType="URL" linkTarget="_blank"

The example below would make the root timeline go to frame number 2
linkData="2" linkType="frameNumber" linkTarget="_root"

The example below would make the root timeline go to the frame with label myFrameLabel
linkData="myFrameLabel" linkType="frameLabel" linkTarget="_root"

The example below would make the movieclip myMovieClip in the root timeline go to frame number 2
linkData="2" linkType="frameNumber" linkTarget="myMovieClip"

The example below would make _root.someMovieClip load someSwf.swf:
linkData="someSwf.swf" linkType="loadMovie" linkTarget="someMovieClip"

Resizing the carousel

To resize the carousel there are a few simple actions you should walk through:

  • Open index.html
  • On line 23 change "1000" and "650" to the desired width and height

Now if you open index.html in your browser, nothing has changed to the size of the carousel orbit itself, only to the stage's size. To change the carousel's size, do the following:

  • Open data.xml
  • Change the centerX and centerY values to the x and y value you want the center of your carousel to be. If you want the carousel to be centered on your stage, these values should be half of the width and height value you entered above.
  • Change the spanX and spanY value to the desired width and height of your carousel. Note that these are the values for the orbit, so the carousel will be slightly bigger because of the images' width and height.
